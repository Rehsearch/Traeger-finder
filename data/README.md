# data/plz-coordinates.json

Deutsche Postleitzahlen mit geografischem Zentrum als Koordinaten (Lat/Lng).

**Quelle:** https://github.com/WZBSocialScienceCenter/plz_geocoord
**Lizenz:** Apache License 2.0 (siehe PLZ-COORDINATES-LICENSE.txt in diesem Ordner)

Ersetzt seit Juli 2026 die vorherige Positionstack-API-Anbindung in
app/api/geocode/route.js — kein externer API-Aufruf mehr nötig für die
PLZ-zu-Koordinaten-Umrechnung, kein zusätzlicher Auftragsverarbeiter.

Datenstand: Download Juli 2026. Sollte gelegentlich (z.B. jährlich)
aktualisiert werden, falls sich PLZ-Gebiete ändern (selten).
